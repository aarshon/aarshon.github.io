    HMI Subsystem for STEM-Based Weather Station Report: Bill of Materials & Component Selection

    This HMI subsystem is designed to provide a robust, low-power, and user-friendly interface for a STEM-based weather station, featuring a PIC18F47Q10 microcontroller, an I²C character LCD, a 4×4 membrane keypad, and a high-efficiency switching regulator. By consolidating essential functionalities—data display, user input, and modular expansion—this solution offers a compact yet scalable platform suitable for educational and real-world applications.

    Rationale

    Microcontroller: PIC18F47Q10-I/PT

    • Key Features: 8-bit MCU, abundant I/O, integrated I²C/SPI, and ultra-low-power modes -Why Chosen: Delivers sufficient processing for keypad scanning, LCD control, and expansion. Surface-mount package saves PCB space.

    LCD Display (EA_DOGM204W-A)

    • Key Features: 20×4 character display, integrated I²C controller, low-power operation -Why Chosen: Clear visual output for weather data and menus. Straightforward I²C interface reduces wiring complexity.

    Power Line Selector (90120-0123)

    • Key Features: 3-pin header/jumper for input source selection -Why Chosen: Allows easy routing of 12 V input from a wall adapter or alternate power source if needed in future expansions.

    4×4 Membrane Keypad (Adafruit #3844)

    • Key Features: 16-button matrix, minimal footprint, plug-and-play connector Why Chosen: Simple scanning approach for user inputs (e.g., up/down/left/right), supported by external interrupts on the MCU.

    Switching Regulator (AP63203WU-7)

    • Key Features: 3.3 V output, up to 2 A load, high efficiency (synchronous buck) Why Chosen: Efficiently steps down the 12 V, 15 A source for all logic-level components, minimizing heat and maximizing battery or adapter efficiency.

    Upstream/Downstream Headers (70246-0801)

    • Key Features: Low-profile, shrouded connectors rated for power and signal lines Why Chosen: Provide robust mechanical connection and easy I²C pass-through for sensor or module expansions.

    Conclusion

    With this set of carefully chosen components, the HMI subsystem is both easy to use and highly adaptable for weather station projects. The PIC18F47Q10 microcontroller provides a powerful but straightforward foundation, while the I²C LCD and 4×4 membrane keypad let users navigate and manage features—like solar panel control—with minimal wiring. The AP63203WU-7 switching regulator cleanly converts the 12 V, 15 A supply down to a stable 3.3 V level, ensuring reliable performance for every part of the system. Expansion headers make it simple to add or swap in new modules, so the design can grow alongside project needs. Overall, this hardware setup meets STEM requirements by balancing reliability, efficiency, and ease of assembly.
